# 구현 내용

### lotto 조건
 - 숫자 범위: 1 - 45
 - 당첨 
   - 1등: 6개 일치 / 2,000,000,000원
   - 2등: 5개 + 보너스 일치 / 30,000,000원
   - 3등: 5개 일치 / 1,500,000원
   - 4등: 4개 일치 / 50,000원
   - 5등: 3개 일치 / 5,000원
 - 로또 1장 가격: 1,000원
 - 구입 금액을 입력받으면 그만큼 로또 발행: 입력 금액 / 1,000원
 
 - 입력
   - 구입 금액
   - 당첨 번호
   - 보너스 번호
 - 출력
   - 로또 수량 및 random 번호 출력
   - 로또 번호와 당첨 번호를 비교하여 수익률 출력 -> 게임 종료
     - 수익률은 소수덤 둘째 자리에서 반올림
   - 예외사항은 [ERROR]로 시작
